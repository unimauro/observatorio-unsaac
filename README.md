# Observatorio UNSAAC

Portal ciudadano de transparencia de la **Universidad Nacional de San Antonio Abad del Cusco** (UNSAAC, pliego 511). Solo datos públicos.

**Live:** https://unimauro.github.io/observatorio-unsaac/

Módulos: presupuesto y ejecución (MEF/SIAF), en qué se gasta cada sol, proveedores y dueños (OECE/CONOSCE), investigación (OpenAlex), gasto↔valor, personal (Portal de Transparencia). Nunca nombres de estudiantes.

Parte de la red de observatorios: [UNI](https://unimauro.github.io/observatorio-uni/) · [San Marcos](https://unimauro.github.io/observatorio-sanmarcos/) · [ENSAD](https://unimauro.github.io/observatorio-ensad/). No es un portal oficial. Correcciones: carlos@cardenas.pe
